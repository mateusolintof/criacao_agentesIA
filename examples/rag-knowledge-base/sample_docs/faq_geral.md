# FAQ - Perguntas Frequentes

Respostas para as perguntas mais comuns sobre nossos produtos e serviços.

## Geral

### Quais formas de pagamento vocês aceitam?

Aceitamos as seguintes formas de pagamento:
- **Cartão de Crédito**: Visa, Mastercard, American Express, Elo
- **Boleto Bancário**: Geração automática mensal
- **Transferência/PIX**: Para contratos anuais
- **Nota de Empenho**: Órgãos públicos

O pagamento é processado no início de cada ciclo. Para planos anuais, oferecemos desconto de 10-15%.

### Posso cancelar a qualquer momento?

Sim! Não há fidelidade. Você pode cancelar a qualquer momento através do painel administrativo ou entrando em contato com o suporte.

**Importante:**
- Cancelamento imediato: válido até o fim do período pago
- Sem cobranças adicionais
- Dados mantidos por 30 dias para recuperação
- Exportação de dados disponível antes do cancelamento

### Vocês oferecem trial gratuito?

Sim! Todos os produtos têm trial gratuito:
- **CRM Enterprise**: 14 dias
- **AI Assistant**: 14 dias + 100k tokens grátis
- **Analytics Suite**: 14 dias

**No trial você tem acesso a:**
- Todas as funcionalidades (versão completa)
- Sem limite de usuários
- Suporte completo
- Sem necessidade de cartão de crédito

### Como funciona o suporte técnico?

Oferecemos suporte em três canais:

**Email** (suporte@empresa.com):
- SLA: 24h (dias úteis)

### Meus dados são usados para treinar os modelos?

**NÃO!** Garantimos total privacidade:

- **OpenAI/Anthropic**: Não armazena nem treina com seus dados
- **Nossa plataforma**: Não usa seus dados para treinar modelos
- **Fine-tuning**: Apenas seus modelos personalizados têm acesso aos dados de treinamento que você fornece
- **LGPD**: Totalmente conforme
- **Contrato DPA**: Disponível para Enterprise

### Qual o limite de tokens?

**Tokens inclusos por plano:**
- Básico (R$ 499/mês): 1 milhão/mês
- Pro (R$ 999/mês): 5 milhões/mês
- Enterprise: Ilimitado

**Tokens adicionais:**
- R$ 50 por 100k tokens
- Compra sob demanda ou pacotes

**Consumo médio:**
- Conversa típica: 500-1000 tokens
- Processamento de documento: 2000-5000 tokens
- Geração de conteúdo: 500-2000 tokens

**Dica**: Use modelos menores (GPT-3.5, Claude Haiku) para reduzir custos sem perder qualidade em tarefas simples.

### Posso usar meus próprios modelos?

Sim! Temos três opções:

**1. Fine-tuning (Recomendado)**
- Use nossos modelos base
- Treine com seus dados
- Hospedagem gerenciada

**2. BYO Model (Bring Your Own)**
- Upload de modelos Llama, Mistral, etc
- Hosting na nossa infraestrutura
- Custo adicional de GPU

**3. API Proxy**
- Use sua própria API key (OpenAI, Anthropic)
- Apenas funcionalidades extras (RAG, analytics)
- Sem cobrança de tokens

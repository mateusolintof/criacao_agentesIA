# Templates

Este diretório contém templates reutilizáveis para acelerar o desenvolvimento de novos projetos de Agentes de IA.

## Estrutura

```
templates/
├── agentes/          # Templates de código para agentes
├── fluxos/           # Templates de fluxos conversacionais
├── integracao/       # Templates de integração com APIs
├── prompts/          # Templates de prompts
├── reunioes/         # Templates de reuniões e workshops
├── personas/         # Templates de personas
├── planejamento/     # Templates de planejamento
├── validacao/        # Templates de validação e testes
└── monitoramento/    # Templates de dashboards e alertas
```

## Como Usar

1. Copie o template relevante para seu projeto
2. Preencha as seções marcadas com `[PREENCHER]`
3. Adapte conforme necessário
4. Remova comentários e instruções antes de finalizar

## Templates Disponíveis

### Planejamento
- `planejamento/escopo.md` - Documento de escopo do projeto
- `planejamento/requisitos.md` - Levantamento de requisitos
- `reunioes/kickoff.md` - Agenda de reunião de kickoff

### Design
- `personas/template-persona.md` - Template de persona
- `fluxos/template-fluxo.md` - Template de fluxo conversacional
- `prompts/template-prompt.md` - Template de prompt

### Desenvolvimento
- `agentes/base_agent.py` - Classe base de agente
- `agentes/sales_agent.py` - Exemplo de agente de vendas
- `integracao/template-spec-api.md` - Especificação de integração

### Validação
- `validacao/formulario-feedback.md` - Formulário de feedback
- `validacao/plano-testes.md` - Plano de testes

### Monitoramento
- `monitoramento/dashboards.json` - Configuração de dashboards
- `monitoramento/alertas.yaml` - Configuração de alertas
